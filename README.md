
# Task 04 – Login Authentication System

## 📌 Project Overview
This project is a simple **Login Authentication System** developed as part of the **Level 2 – Task 04** of my Web Development and Designing Internship.

The system allows users to:
- Register a new account
- Login with their credentials
- Access a secure page after successful login

This project demonstrates the basic concept of **user authentication and secure access control** using web technologies.

---

## 🎯 Objective
The objective of this task is to build a simple authentication system that verifies user credentials and restricts access to protected pages unless the user is logged in.

---

## 🛠 Technologies Used
- **HTML5** – Structure of the web pages
- **CSS3** – Styling and layout design
- **JavaScript** – Authentication logic and validation

---
## Features
- User Registration System
- Login Authentication
- Secure Page Access
- Form Validation
- Simple and Responsive UI

---

## Learning Outcome
Through this project, I learned:
- Basic authentication flow
- Form handling in JavaScript
- Creating secure page access
- Connecting HTML, CSS, and JavaScript files
